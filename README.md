## 啟動程式
1. 將OpenAI API KEY 放進 .env.example
``` bash
OPENAI_API_KEY = 你的API KEY
```
2. 將.env.example 改名成 .env
3. 跑以下指令
``` bash
pip install -r requirements.txt
```
4. 使用 uvicorn 啟動 fastAPI
